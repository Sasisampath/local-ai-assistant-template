# Local AI Assistant Template

Run a private chatbot fully on your machine using TinyLlama.

This template helps builders launch a lightweight local AI assistant without relying on external APIs or cloud infrastructure.

Perfect for experimenting with:

• offline AI assistants
• private copilots
• internal tools
• learning LLM architecture
• secure local-first workflows

---

## Features

• Runs entirely on your machine  
• Powered by TinyLlama 1.1B  
• Session-based contextual memory  
• Password-protected access  
• No permanent conversation storage  
• Simple Flask backend architecture  
• Example GPT-style model initialization script included  

---

## Use Cases

Start building:

• private journaling assistants  
• offline copilots  
• internal team tools  
• research helpers
